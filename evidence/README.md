# Omnis Go-Live Evidence Log

Companion folder to `OMNIS_GO_LIVE_TESTING_AND_ACCREDITATION_CHECKLIST.md`.
Rule from that document: don't tick a checkbox until there's a file in here
proving it — a log, a screenshot, a scan report, a test transcript. Each
phase subfolder's README lists exactly what's expected and which checklist
item it satisfies.

| Folder | Checklist phase |
|---|---|
| `phase5-mis-synthetic-data/` | Phase 5 — MIS Integration & Synthetic Data Testing |
| `phase6-load-resilience/` | Phase 6 — Load, Resilience & Failure Testing |
| `phase7-security/` | Phase 7 — Security Testing & Certification |
| `phase8-childrens-code/` | Phase 8 — Data Protection & Children's Code Compliance |
| `phase9-accreditation/` | Phase 9 — External Accreditation & Evaluation |
| `phase10-operational/` | Phase 10 — Operational Go-Live Readiness |

When a piece of evidence lands, update the corresponding checkbox in the
checklist and the Status Tracker table at the top of it — the checklist and
this folder should never drift apart.
